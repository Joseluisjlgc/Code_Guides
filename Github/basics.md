GITHUB BASICS
==============

# WINDOWS

 ## Create repository
  - Go to Source Control(CTRL+Shift+G)
  - Initialize Repository

 ## Change name of branch
  - Go to the search bar(Shift+CTRL+P)
  - Type "rename branch"
  - Type new name

 ## Commit
  - Stage (git add --all or click the "+"  signs in the files on SOURCE CONTROL)
